# Quiz Game With Timer

This project is a static HTML/CSS/JavaScript quiz game with:
- Login
- Registration
- Timed quiz
- Leaderboard

## Run Locally

Use any static server, then open `login.html`.

Example:

```powershell
py -m http.server 8080
```

Open:

`http://localhost:8080/login.html`

## Deploy On GitHub Pages

1. Create a new GitHub repository.
2. Upload all project files from this folder.
3. In GitHub repo: `Settings` -> `Pages`.
4. Under `Build and deployment`:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Save and wait 1-2 minutes.
6. Open your site URL shown in Pages settings.
7. Open `.../login.html` to start from login screen.

## Device Support

The UI is responsive and supports:
- Mobile phones
- Tablets
- Laptops
- Desktop computers
- MacBook/iMac

For best results, use latest Chrome, Edge, Safari, or Firefox.
